# MKDD: The Elite Persuit

Just another custom tracks distribution for Mario Kart Double Dash.

Grand vision is to include an online mode and a normal mode to play at your leasure.


## Software Requirements

- **Mega CMD**: Since people like using MEGA for track uploads. Install locally just make sure its in the computer's PATH.

## Project Structure

- **bin/**: Contains dependencies used by the project. Such as Gecko Loader and the MKDD Extender.
- **track_downlaoder/**: Source code for the track downloader files.
- **tracks/**: Includes tracks used by the project, separated into "tracks/<track_name> (<version>)".
